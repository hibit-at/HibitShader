# Abstract

**Hibit Shader** is an alternative Unity shader made by [@hibit_at](https://twitter.com/hibit_at)

# Features

- Easy to set up (only 2 parameter)
- Make your avatar lit up softly
- Normal map available
- Hue changing system.
